# Olá, bem-vindo ao perfil oficial da GestPro! 👋

Este é o espaço onde centralizamos o desenvolvimento e a inovação por trás da **GestPro**, uma solução robusta e intuitiva para a gestão de pequenos e médios negócios.

---

## 🚀 Sobre a GestPro

A **GestPro** nasceu com a missão de substituir planilhas complexas e anotações manuais por um controle digital estratégico. Nosso foco é transformar dados em decisões, oferecendo ferramentas de PDV, controle de estoque e dashboards financeiros em uma única plataforma.

### 🛠️ Nosso Ecossistema Tecnológico

Desenvolvemos utilizando o que há de mais moderno para garantir performance, segurança e escalabilidade:

* **Frontend:** Next.js 14+, TypeScript, Tailwind CSS e shadcn/ui.
* **Backend:** Java 17+, Spring Boot 3.x, Spring Security (JWT & OAuth2).
* **Banco de Dados:** MySQL para persistência e Redis para alta performance.
* **Infraestrutura:** Docker para padronização de ambientes e deploys otimizados.

---

## 📈 Projeto

### [Gest-Pro](https://github.com/Gest-profissional/Gest-Pro)
O core da nossa plataforma. Um sistema de gestão completo com:
- **Frente de Caixa (PDV):** Registro de vendas ágil e intuitivo.
- **Gestão de Estoque:** Alertas de estoque baixo e histórico de movimentações.
- **Dashboards Inteligentes:** Visão clara de lucros e performance do negócio.
- **Segurança Avançada:** Autenticação via Google e proteção de dados de ponta a ponta.
