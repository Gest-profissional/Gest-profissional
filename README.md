# GestPro: Gestão Inteligente

Solução digital estratégica desenvolvida para transformar o controle de pequenos e médios negócios, substituindo a desordem por dados reais.

---

### 🛠️ Tech Stack
* **Frontend:** Next.js 14 (App Router), TypeScript & Tailwind CSS.
* **Backend:** Java 17, Spring Boot 3 & Spring Security (JWT).
* **Infra:** MySQL, Redis & Docker.

---

### 📦 O Projeto
O [**Gest-Pro**](https://github.com/Gest-profissional/Gest-Pro) centraliza o essencial para o negócio em um único lugar:
* **PDV:** Registro de vendas ágil e intuitivo.
* **Estoque:** Controle de entrada, saída e alertas inteligentes.
* **Dashboards:** Visão clara de performance e lucratividade.
* **Segurança:** Autenticação via Google e proteção de dados.
